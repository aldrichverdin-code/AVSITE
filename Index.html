Educia

Aplicación móvil y web desarrollada con React Native Expo y React utilizando marcos como Bootstrap y CSS puro. Proyecto 04
Agosto 2025

MayorStore

Aplicación web desarrollada con React, utilizando Bootstrap y CSS para la interfaz de usuario y Django para el back-end. Se creó una base de datos con PostgreSQL, en la que se implementaron API y sus respectivas llamadas, con varios niveles de usuario. El proyecto consiste en un sistema de seguimiento de pedidos de textiles, en el que los pedidos de ropa se realizan a través de la interfaz de usuario, mientras que el control de inventario y la gestión de pedidos se llevan a cabo en la interfaz de administración.
